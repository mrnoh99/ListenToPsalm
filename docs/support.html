시편듣기 — VoiceOver · 시리 · 손쉬운 사용 안내

시각장애인 · VoiceOver(화면 읽기) 사용 안내

시편듣기는 Apple VoiceOver를 지원합니다. 화면을 보지 않고도 아래 기능을 음성 안내로 이용할 수 있습니다.

  • 시편 선택: 마태오·마르코·루카·요한 버튼 이름과 선택 상태 읽기
  • 장 목록: 「마태오복음서 삼장」처럼 장 번호를 일·이·삼으로 읽고, 재생 중·이어보기 진행률을 말로 안내
  • 재생·정지: 「마태오복음서 오장 재생 버튼」 / 「… 중지 버튼」처럼 복음·장·동작을 한꺼번에 읽음
  • 수면 타이머: 「타이머 버튼」으로 읽음. 바 왼쪽 복음서 이름은 VoiceOver 포커스에서 제외
  • 상태 알림: 재생 시작·정지·시편 변경·타이머 변경 시 햅틱·짧은 시스템 소리
  • 탐색 순서: 앱 제목 → 시편 그리드 → 수면 타이머 바 → 재생 바 → 장 목록
  • 시리·단축어: 말로 장 재생·이어 듣기·수면 타이머·정지 (안내). 시리 사용 후에도 재생이 자동으로 이어짐

켜는 방법

iPhone에서 설정 → 손쉬운 사용 → VoiceOver를 켠 뒤 앱을 실행하세요. 한 손가락으로 화면을 오른쪽으로 밀면 다음 항목을, 두 번 탭하면 선택합니다.

VoiceOver 사용에 익숙하지 않으시면 Apple의 iPhone VoiceOver 사용 설명서도 참고해 주세요.

작동법

  1. 시편 고르기: 화면 위쪽 2×2 버튼(마태오·마르코·루카·요한)을 탭합니다.
  2. 장 고르기: 스크롤 목록에서 원하는 장을 탭하면 재생이 시작됩니다.
  3. 재생·정지: 화면 아래 「재생」 캡슐 바를 탭합니다. 한 번 누르면 재생, 재생 중이면 정지합니다.
  4. 수면 타이머: 시편 그리드 바로 아래 글래스 캡슐 바 전체(왼쪽 복음서 이름 포함)를 탭해 시간을 고릅니다.
  5. 이어 듣기: 정지한 뒤 같은 실행에서 「재생」을 누르거나, 앱을 다시 열어 「이어서 … 재생」을 탭합니다.
  6. 잠금 화면: 재생 중에는 잠금 화면·이어폰에서도 재생·일시정지할 수 있습니다.

손쉬운 사용 · 재생 편의

햅틱 피드백

재생·정지·장이 바뀔 때 iPhone이 짧게 진동해, 화면을 보지 않아도 상태 변화를 느낄 수 있습니다. VoiceOver 음성 안내와 함께 쓰면 더 편합니다.

Voice Control(음성 제어)

iPhone 설정 → 손쉬운 사용 → Voice Control을 켠 뒤, 예를 들어 「재생 탭」「정지 탭」「마태오 탭」처럼 말하여 버튼을 누를 수 있습니다. 장 목록은 「마태오복음서 3장 탭」처럼 장 이름으로도 선택할 수 있습니다.

이어 듣기 제안

앱을 종료했다가 다시 열면, 마지막으로 듣던 장이 있을 때 「이어서 마태오 5장 재생」 버튼이 표시됩니다. 한 번 탭하면 저장된 위치에서 이어 듣습니다. 같은 실행 중에 방금 정지한 경우에는 기존 재생 버튼으로 이어 들을 수 있습니다.

잠금 화면 · 제어 센터

재생 중에는 잠금 화면·제어 센터·이어폰의 재생 · 일시정지 제어로 말씀 듣기를 조작할 수 있습니다. 화면을 끈 채로도 이어서 들을 수 있습니다.

시리 · 단축어로 말로 실행

App Intents를 지원합니다. 앱을 한 번 실행한 뒤 아래처럼 말하거나, 단축어 앱에서 「시편듣기」 관련 작업을 추가할 수 있습니다.

복음서 이름: 전체 이름(마태오복음서 · 마르코복음서 · 루카복음서 · 요한복음서) 또는 짧은 이름(마태오 · 마르코 · 루카 · 요한) 둘 다 인식합니다.

중요: 시리 명령에는 반드시 앱 이름 「시편듣기」가 포함되어야 합니다. 앱 이름 없이 자주 사용하는 문구는 단축어 앱에서 본인의 호출 문구를 직접 등록할 수 있습니다.

설정 확인

iPhone 설정 → 시리와 검색 → 시편듣기에서 제안되는 문구를 확인할 수 있습니다. 60·90·120분 수면 타이머는 단축어 앱의 「수면 타이머 설정」에서 분을 선택하세요.

자주 묻는 질문

소리가 나오지 않아요.

기기 음량·무음 스위치·다른 앱 재생 상태를 확인해 주세요. 설정에서 마이크가 아닌 재생 볼륨을 조정해 보세요.

VoiceOver로 어떻게 쓰나요?

iPhone 설정 → 손쉬운 사용 → VoiceOver를 켠 뒤 앱을 실행하세요. 복음·장·재생·수면 타이머가 모두 음성으로 읽히며, 재생·정지 시에도 알림이 들립니다. 위 VoiceOver 안내를 참고해 주세요.

정지 후 다시 재생하면 처음부터 나와요.

정지 시점의 장과 위치를 기억합니다. 다른 복음을 둘러본 뒤 재생을 누르면 정지했던 장에서 이어 들을 수 있습니다. 목록에서 다른 장을 탭하면 그 장부터 새로 재생됩니다.

잠금 화면에서 재생을 조작할 수 있나요?

네. 재생 중 잠금 화면·제어 센터·이어폰의 재생 · 일시정지 제어로 듣기를 조작할 수 있습니다. 장을 직접 바꾸고 싶을 때는 앱을 열어 장 목록에서 선택하거나, 시리에게 「시편듣기에서 마태오 5장 틀어줘」처럼 말씀해 주세요.

앱을 다시 열면 이전에 듣던 곳부터 시작할 수 있나요?

네. 마지막 재생 위치가 저장되어 있으면 「이어서 ○○ 장 재생」 버튼이 나타납니다. 한 번 탭해 이어 듣기. 정지 직후에는 재생 버튼으로도 이어 들을 수 있습니다.

시리나 단축어로 재생할 수 있나요?

네. App Intents를 지원합니다. 앱을 한 번 실행한 뒤 「시편듣기에서 마태오 3장 재생」, 「이어서 재생」, 「수면 타이머 30분」, 「시편듣기 정지」처럼 말할 수 있습니다. 위 시리·단축어 안내 표를 참고해 주세요.

수면 타이머는 어떻게 동작하나요?

시편 그리드 아래 수면 타이머 캡슐 바 전체(왼쪽 「마태오복음서」 글자 영역 포함)를 탭하면 설정 창이 열립니다. 30·60·90·120분 또는 ‘계속’을 고릅니다. 타이머는 재생 중일 때만 만료 시 자동으로 멈추며, ‘계속’이면 밤새 재생됩니다. 남은 시간은 바 오른쪽에 표시됩니다.

시리를 쓰면 「다른 앱의 오디오…」 메시지가 나와요.

예전 버전에서는 시리가 마이크를 쓸 때 잠깐 재생이 멈추며 해당 안내가 표시될 수 있었습니다. 최신 버전에서는 안내 없이 잠시 멈췄다가 시리가 끝나면 자동으로 이어 재생됩니다. 계속 문제가 있으면 앱을 최신 버전으로 업데이트해 주세요.

VoiceOver에서 장 번호가 「둘」이 아니라 「이」로 읽히나요?

네. 성경 장수에 맞게 일·이·삼·사…으로 읽도록 맞춰 두었습니다. 예: 2장 → 「이장」, 12장 → 「십이장」. 재생 버튼·장 목록·이어 듣기 제안에 동일하게 적용됩니다.

iPad나 Mac에서도 되나요?

프로젝트 설정에 따라 지원 대상 기기가 달라질 수 있습니다. App Store의 ‘정보’에서 호환 기기를 확인해 주세요.
