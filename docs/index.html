시편듣기

시각장애인을 위한 VoiceOver(화면 읽기) 지원

시편듣기는 iPhone의 VoiceOver를 지원합니다. 화면을 보기 어려우셔도 복음·장 선택, 재생·정지, 수면 타이머를 음성 안내와 터치만으로 이용하실 수 있습니다.

시편 이름·장 제목·재생 상태·경과 시간을 말로 읽어 주며, 재생 시작·정지·시편 변경·타이머 설정 시에도 음성으로 알려 드립니다. 말씀을 듣는 데 맞춘 앱이라 시각장애인 분께 특히 잘 맞도록 설계했습니다.

사용 방법: iPhone 설정 → 손쉬운 사용 → VoiceOver를 켠 뒤 앱을 실행하세요. 시리·단축어 · Voice Control · 햅틱 등 작동법 · 손쉬운 사용 기능 · 지원 페이지를 참고해 주세요.

손쉬운 사용 · 재생 편의

  • 햅틱 피드백: 재생·정지·장이 바뀔 때 짧은 진동으로 상태를 확인합니다.
  • Voice Control: 「재생 탭」「마태오 탭」처럼 음성만으로 버튼을 누를 수 있습니다.
  • VoiceOver 포커스 순서: 타이틀 → 복음서 그리드(4개) → 수면 타이머 바 → 재생 바 → 장 목록 순으로 탐색됩니다.
  • VoiceOver 명확한 라벨: 수면 타이머는 「타이머 버튼」으로 읽고, 복음서 이름(예: 마태오복음서)은 타이머 바에서 포커스되지 않습니다. 재생 버튼은 「마태오복음서 이장 재생 버튼」처럼 복음·장·동작을 한 번에 읽습니다.
  • 장 번호 읽기: 장 번호는 「하나·둘·셋」이 아니라 성경 장수에 맞게 「일·이·삼…」으로 읽습니다(예: 3장 → 「삼장」).
  • 이어 듣기 제안: 앱을 다시 열면 「이어서 마태오 5장 재생」을 한 번 탭해 이전 위치에서 재개합니다.
  • 잠금 화면·제어 센터: 재생 중 잠금 화면에서 재생·정지를 조작할 수 있습니다.

최근 업데이트

  • 복음서 탭 전환 중 재생 유지: 재생 중에 마태오·마르코·루카·요한 그리드에서 다른 복음서를 눌러도 현재 듣고 있는 장은 끊기지 않고 계속 재생됩니다.
  • 큰 글씨의 복음서 그리드와 재생 버튼: 그리드 크기는 그대로 유지하면서 글자만 크게 키워 가독성을 높였습니다.
  • 타이머 바 전체 터치: 「마태오복음서」 글자 영역을 포함해 수면 타이머 캡슐 바 어디를 눌러도 타이머 설정이 열립니다.
  • 리퀴드 글래스 효과 확장: 장 목록이 타이머·재생 버튼 뒤까지 확장되어, 두 캡슐 모두 아래 콘텐츠가 비치는 동일한 투명도로 표시됩니다.
  • 시리 사용 시 재생 유지: 시리로 명령할 때 잠깐 멈춰도 「다른 앱의 오디오…」 안내 없이, 시리가 끝나면 듣던 장에서 자동으로 이어 재생됩니다.
  • 시리 자연어 명령 확대: 「시편듣기 열어줘」 「시편듣기에서 요한복음서 3장 틀어줘」 「시편듣기 멈춰」 등 한국어 동사·앱 이름 조합을 인식합니다.

작동법

  1. 시편 선택: 상단 2×2 그리드에서 마태오·마르코·루카·요한 중 하나를 탭합니다. 재생 중에 다른 복음을 눌러도 지금 듣는 장은 끊기지 않습니다.
  2. 장 선택: 아래 목록에서 장을 탭하면 그 장부터 재생합니다. 재생 중인 장은 목록에서 강조됩니다.
  3. 재생·정지: 화면 하단 캡슐 바의 재생 버튼을 탭합니다. 재생 중이면 정지, 정지 상태이면 이어 듣기 또는 선택한 장 재생입니다.
  4. 수면 타이머: 시편 그리드 아래 캡슐 바 전체를 탭해 30·60·90·120분 또는 「계속」을 고릅니다. 재생 중에만 만료 시 자동 정지합니다.
  5. 이어 듣기: 앱을 다시 열면 마지막 위치 「이어서 ○○장 재생」 제안이 나타날 수 있습니다.
  6. 시리·단축어: 앱을 한 번 실행한 뒤 「시편듣기에서 마태오 3장 틀어줘」처럼 말합니다. 재생 중 시리를 써도 끝나면 자동으로 이어 재생됩니다.

앱 소개

마태오·마르코·루카·요한 복음을 장별로 선택해 들을 수 있습니다. 이동 중이나 잠들기 전처럼 성경을 펼치기 어려울 때도 말씀에 귀 기울일 수 있도록 단순한 화면과 재생에 집중했습니다.

시리와 단축어로도 이용할 수 있습니다. 앱을 한 번 실행한 뒤 「시편듣기 마태오 3장 재생」, 「시편듣기 이어 들려줘」, 「시편듣기 수면 타이머 30분」처럼 말로 재생을 시작하거나 멈출 수 있습니다.

주요 기능

  • VoiceOver 지원: 복음·장·재생·수면 타이머를 음성으로 탐색·조작
  • 시리·단축어: 말로 장 재생, 이어 듣기, 수면 타이머, 정지
  • Voice Control: 「재생 탭」「마태오 탭」 등 음성 명령으로 조작
  • 햅틱·이어 듣기 제안·잠금 화면 재생 제어 (자세히)
  • 4복음서중 원하는 복음과 장 선택 후 재생
  • 선택한 장부터 시편 끝까지 연속 재생
  • 지금 재생 중인 장을 목록에서 쉽게 확인
  • 정지 후 재생 시, 가능한 경우 이전에 듣던 위치에서 이어 듣기
  • 수면 타이머: 잠자리에서 말씀 듣기에 맞춰 30·60·90·120분 뒤 자동으로 멈추거나, 밤새 들으려면 ‘계속’을 선택하세요. 남은 시간이 화면·음성 안내로 표시됩니다.
